# IT2 Eksamen

## Om prosjektet

Dette prosjektet er utviklet som del av praktisk eksamen i Informasjonsteknologi VG2.

## Teknologier

* Node.js
* Express
* MySQL
* HTML/CSS/JavaScript

## Funksjoner

* Innlogging
* Rollebasert tilgang
* Database med CRUD-operasjoner
* Administrasjonspanel

## Starte prosjektet

Installer dependencies:

```bash
npm install
```

Start server:

```bash
npm start
```
